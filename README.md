# Features 

* Supports HTML Elements,Function,String,Variable
* To navigate up one directory level, use ../dir
* Not Case Sensitive with String
* Colorful Output 


# How to install and use (* linux)
* git clone https://github.com/c0brabaghdad1/Iv0ry.git
* cd Iv0ry
* chmod +x iv0ry.awk 
* ./iv0ry.awk
